Título y descripción: AeroViajes, es un proyecto académico cuya función se centra en la simulación de reserva, compra y venta de pasajes de vuelos nacionales e internacionales a través de una plataforma sencilla e intuitiva.
El código se ha desarrollado en el lenguaje de programación Java dentro de Intellij IDEA.
Del mismo modo, el código está conectado a una base de datos desarrollada en MySQL.
La idea de desarrollar esta aplicación nace como puesta en práctica de los conocimientos adquiridos dentro del ámbito académico.
